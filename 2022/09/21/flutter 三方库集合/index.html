三方库合集

cupertino_icons: ^1.0.2:

Flutter 的Cupertino 小部件使用的默认图标资产集。

https://pub.dev/packages/cupertino_icons
get: ^4.6.5:

高性能状态管理、智能依赖注入和快速实用的路由管理。

https://pub.dev/packages/get
url_launcher: ^6.1.5:

启动 URL 的 Flutter 插件。

https://pub.dev/packages/url_launcher
image_picker: ^0.8.5+3:

从图像库中挑选图像，并使用相机拍摄新照片。

https://pub.dev/packages/image_picker
path_provider: ^2.0.11:

查找文件系统上常用位置的 Flutter 插件。

https://pub.dev/packages/path_provider
shared_preferences: ^2.0.15:

为简单数据包装特定于平台的持久存储。数据可能会异步持久化到磁盘，并且不能保证写入返回后会持久化到磁盘，所以这个插件一定不能用于存储关键数据。
支持的数据类型是int、double、bool和。StringList

https://pub.dev/packages/shared_preferences
webview_flutter: ^3.0.4:

一个提供 WebView 小部件的 Flutter 插件。

https://pub.dev/packages/webview_flutter
flutter_local_notifications: ^11.0.0:

显示本地通知的跨平台插件。

https://pub.dev/packages/flutter_local_notifications
flutter_screenutil: ^5.5.4:

适配屏幕和字体大小的flutter插件。

https://pub.dev/packages/flutter_screenutil
permission_handler: ^10.0.0:

请求权限并检查其状态。还可以打开设备的应用设置，以便用户授予权限。

https://pub.dev/packages/permission_handler
flutter_native_splash: ^2.2.9:

自动生成 iOS、Android 和 Web 原生代码，用于自定义此原生闪屏背景颜色和闪屏图像。支持暗模式、全屏和特定于平台的选项。

https://pub.dev/packages/flutter_native_splash
package_info_plus: ^1.4.3+1:

提供了一个用于查询应用程序包信息的 API。

https://pub.dev/packages/package_info_plus
connectivity_plus: ^2.3.7:

允许 Flutter 应用程序发现网络连接并进行相应的配置。可以区分蜂窝连接和 WiFi 连接。

https://pub.dev/packages/connectivity_plus
logger: ^1.1.0:

小型、易于使用和可扩展的记录器，可打印漂亮的日志。

https://pub.dev/packages/logger
dio: ^4.0.6:

一个强大的 Dart Http 客户端，支持拦截器、全局配置、FormData、请求取消、文件下载、超时等。

https://pub.dev/packages/dio
font_awesome_flutter: ^10.2.1:

Font Awesome图标包作为 Flutter 图标集提供
此图标包仅包含Font Awesome 开箱即用的免费图标

https://pub.dev/packages/font_awesome_flutter
getwidget: ^3.0.1:

GetWidget是一个 100% 免费的 Flutter 开源 UI Kit 库，使用Flutter SDK构建。GetWidget 有 1000 多个预构建的小部件。

https://pub.dev/packages/getwidget
pull_to_refresh: ^2.0.0:

提供给flutter滚动组件下拉刷新和上拉load的一个widget。支持android和ios。

https://github.com/peng8350/flutter_pulltorefresh/blob/master/README_CN.md
device_info_plus: ^4.1.2:

从 Flutter 应用程序中获取当前设备信息。

https://pub.dev/packages/device_info_plus
qr_code_scanner: ^1.0.1:

通过在 Flutter 中原生嵌入平台视图，可在 iOS 和 Android 上运行的 QR 码扫描器。这个插件不再是最新的并且只处于维护模式。
mobile_scanner: ^2.0.0：

qr_code_scanner替代方案

https://pub.dev/packages/mobile_scanner
`image_cropper: ^3.0.0:

适用于 Android、iOS 和 Web 的 Flutter 插件支持裁剪图像。该插件基于三个不同的本地库，因此在这些平台之间具有不同的 UI。

https://pub.dev/packages/image_cropper
flutter_colorpicker: ^1.0.3:

HSVHSB/HSL/RGB/Material 颜色选择器的灵感来自于您惊人的颤振应用程序的所有优秀设计。
开箱即用的可爱颜色选择器，具有高度定制的小部件，可满足所有开发人员的需求。

https://pub.dev/packages/flutter_colorpicker
syncfusion_flutter_datepicker: ^20.2.50:

一个轻量级的小部件，允许用户轻松选择单个日期、多个日期或一个日期范围。日期选择器提供月、年、十年和世纪视图选项，以快速导航到所需的日期。它支持最小、最大、中断和禁用日期以限制日期选择。https://pub.dev/packages/syncfusion_flutter_datepicker
day_night_time_picker: ^1.1.3:

零依赖的 Flutter 日夜时间选择器。

https://pub.dev/packages/day_night_time_picker
city_pickers: ^1.1.3:

中国的城市三级联动选择器

https://pub.dev/packages/city_pickers
demo: https://pmbee.alicdn.com/flutter/city_pickers/index.html
